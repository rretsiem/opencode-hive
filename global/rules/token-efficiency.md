# Token Efficiency

- Use the `skeleton` tool before reading large Python files — understand the surface first
- Use the `impact` tool before renaming or changing any symbol — know the blast radius
- Use the `which_test` tool to find relevant tests before and after changes
- Prefer native tools (read, grep, glob) over bash equivalents (cat, rg, find)
- When delegating to specialists, provide exact file paths — don't make them search
- Keep responses concise — direct output, not verbose explanations
